# InstaVibe Multi-Agent AI System

A multi-agent AI application built with Google ADK, A2A, MCP, Cloud Run, Vertex AI Agent Engine, and Cloud Trace.

## 🌐 Live Demo

🔗 https://instavibe-809444069363.us-central1.run.app

The application is deployed on Google Cloud Run.

## 🚀 Project Overview

InstaVibe is a multi-agent AI system built on Google Cloud using:

- Google ADK
- A2A (Agent-to-Agent Communication)
- MCP (Model Context Protocol)
- Vertex AI Agent Engine
- Cloud Run
- Cloud Trace

The system demonstrates how multiple AI agents can collaborate, communicate, and orchestrate tasks in a cloud-native environment.

---

## 🧠 Multi-Agent Architecture

This project includes multiple specialized agents:

- Planner Agent
- Social Agent
- Platform Agent
- Orchestrator Agent

The Orchestrator coordinates the communication between agents using A2A protocols.
